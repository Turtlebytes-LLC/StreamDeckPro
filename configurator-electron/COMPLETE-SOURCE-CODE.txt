QUICK START
================================================================================

1. Create the two config files above (tailwind.config.js and postcss.config.js)

2. For src/renderer.js and src/styles.css:
   - If you want me to generate them, ask: "please create the renderer and styles files"

3. Once all files are created:
   cd configurator-electron
   npm run dev     # Terminal 1
   npm start       # Terminal 2

4. The configurator will open and you can start configuring your Stream Deck!

================================================================================
IMPORTANT NOTES
================================================================================

- The renderer.js file uses vanilla JavaScript (no framework)
- All state management is simple and straightforward
- File operations go through the Electron IPC (window.api.*)
- The UI automatically reloads configurations on startup
- Toast notifications provide feedback for all actions
- Dark mode preference is saved to localStorage

================================================================================
ARCHITECTURE
================================================================================

main.js (Electron Main Process)
  ↕ IPC Communication
preload.js (Secure Bridge)
  ↕ window.api.*
renderer.js (Frontend Logic)
  → index.html (UI Structure)
  → styles.css (Tailwind Styling)

All file operations:
renderer.js → window.api.readFile() → preload.js → main.js → fs.readFile()
